位置尺度與座標軸

概覽

ggplot2 產生的每一張圖都仰賴位置尺度（position scales），把資料值映射到繪圖表面上的實際位置。x 與 y 美學各自都有一個相關聯的位置尺度，即使使用者沒有明確宣告它們。

考慮這個最簡的直方圖規格：

ggplot(mpg, aes(x = displ)) + geom_histogram()

y 美學從未被提及，但圖仍正確呈現，因為 geom_histogram() 在內部計算了一個 count 變數並把它映射到 y。完全明確的等價寫法是：

ggplot(mpg, aes(x = displ, y = after_stat(count))) + geom_histogram()

由於位置尺度出現在每一張圖中，透徹理解它們的運作方式——以及如何被客製化——對於有效的資料視覺化至關重要。

本章組織成四個主要章節：

  • §10.1——數值（連續）位置尺度：範圍界限、縮放、擴展、刻度、次刻度、標籤與轉換
  • §10.2——日期時間位置尺度：專門的刻度與標籤格式
  • §10.3——離散位置尺度：重新排序、自訂標籤與軸指引
  • §10.4——分箱位置尺度：把連續變數切成離散的箱

數值位置尺度

數值位置尺度

預設的連續位置尺度 scale_x_continuous() 與 scale_y_continuous()，建立了一個從原始資料值到繪圖面板上像素座標的線性映射。

一族便利包裝函式——scale_x_log10()、scale_x_reverse()、scale_x_sqrt() 及其 y 軸對應——提供對最常需要之轉換的快速存取（在 §10.1.7 完整說明）。

所有連續位置尺度的基本行為都由六個關鍵引數支配：

引數 用途
limits 定義尺度的範圍
expand 在資料之外加上填補
breaks 設定主刻度的位置
minor_breaks 設定次刻度的位置
labels 控制刻度標籤文字
trans 套用一個數學轉換

範圍界限

limits 引數控制連續尺度被定義的範圍。預設情況下，ggplot2 自動從資料範圍推導出界限，但在跨多張圖比較時，往往需要手動指定。

library(ggplot2)
library(dplyr)

# mpg 的兩個子集，它們會有不同的自然軸範圍
mpg_99 <- mpg |> filter(year == 1999)
mpg_08 <- mpg |> filter(year == 2008)

base_99 <- ggplot(mpg_99, aes(displ, hwy)) + geom_point()
base_08 <- ggplot(mpg_08, aes(displ, hwy)) + geom_point()

沒有明確的界限，每張圖都使用自己的自然資料範圍，使並排比較不可靠。

要同步座標軸，在每個尺度中傳一個長度為 2 的數值向量給 limits。如果你只想限制一端，用 NA 作為佔位：

library(patchwork)
p99 <- base_99 +
  scale_x_continuous(limits = c(1, 7)) +
  scale_y_continuous(limits = c(10, 45))
p08 <- base_08 +
  scale_x_continuous(limits = c(1, 7)) +
  scale_y_continuous(limits = c(10, 45))
p99 | p08

lims() 便利函式在同時為多個美學設定界限時減少冗長：

p99b <- base_99 + lims(x = c(1, 7), y = c(10, 45))
p08b <- base_08 + lims(x = c(1, 7), y = c(10, 45))
p99b | p08b

lims() 遵循與 labs() 相同的名稱–值語法：引數名稱指定美學，而值指定其界限。單軸的輔助函式 xlim() 與 ylim() 也可用於快速調整。

放大

縮窄尺度的 limits 並不等同於視覺上的放大。當一個資料值落在連續尺度所宣告的界限之外時，ggplot2 會在計算統計摘要之前把它轉換成 NA。這可能無聲地扭曲諸如 stat_boxplot 等 stats 的輸出。

base <- ggplot(mpg, aes(drv, hwy)) +
  geom_hline(yintercept = 28, colour = "red") +
  geom_boxplot()
base

比較限制可見範圍的兩種方法：

# 正確：放大而不捨棄資料
p_zoom <- base + coord_cartesian(ylim = c(10, 35))

# 錯誤：在計算 stat 之前捨棄了資料
p_lim  <- base + ylim(10, 35)   # 警告：列被移除

p_zoom | p_lim

左圖安全地放大。右圖則在被截斷的樣本上無聲地重新計算了盒鬚圖，把中位數往下移。當意圖純粹是視覺上的放大時，務必使用 coord_cartesian()。

視覺範圍擴展

預設情況下，ggplot2 會在資料界限之外加上一個小緩衝，使所繪物件不會與軸線相撞。這個緩衝透過 expand 引數控制，它接受一個 expansion() 呼叫。

base_raster <- ggplot(faithfuld, aes(waiting, eruptions)) +
  geom_raster(aes(fill = density)) +
  theme(legend.position = "none") +
  labs(x = NULL, y = NULL)

# 移除所有填補（對光柵／影像圖很有用）
base_raster +
  scale_x_continuous(expand = expansion(0)) +
  scale_y_continuous(expand = expansion(0))

expansion() 接受兩個引數：

  • add——以資料單位表示的固定空間量（與變數同尺度）
  • mult——相對於完整軸範圍的比例量

兩者都可為長度 1（對稱）或長度 2（不對稱：下、上）。

base_raster +
  scale_x_continuous(expand = expansion(add = 3)) +
  scale_y_continuous(expand = expansion(add = 3))
# 四邊各 20% 的乘法填補
base_raster +
  scale_x_continuous(expand = expansion(mult = .2)) +
  scale_y_continuous(expand = expansion(mult = .2))
# 不對稱：低端 5%，高端 20%
base_raster +
  scale_x_continuous(expand = expansion(mult = c(.05, .2))) +
  scale_y_continuous(expand = c(.05, 0, .2, 0))

刻度

ggplot2 中的軸刻度標記是尺度刻度（scale breaks）的一個特例。它們的位置透過相關尺度函式的 breaks 引數設定。

# 本節全程使用的玩具資料集
toy <- data.frame(
  const = 1,
  up    = 1:4,
  txt   = letters[1:4],
  big   = (1:4) * 1000,
  log   = c(2, 5, 10, 2000)
)

base <- ggplot(toy, aes(big, const)) +
  geom_point() +
  labs(x = NULL, y = NULL) +
  scale_y_continuous(breaks = NULL)   # 抑制 y 軸刻度
base

設定 breaks = NULL 會移除刻度標記以及相關的主格線。

手動指定一個刻度位置的向量，會同時移動主格線與位於它們之間的次格線：

p1 <- base + scale_x_continuous(breaks = c(1000, 2000, 4000))
p2 <- base + scale_x_continuous(breaks = c(1000, 1500, 2000, 4000))
p1 / p2

breaks 引數也接受一個函式，它接收尺度的界限並回傳刻度位置。scales 套件提供數個現成的刻度函式：

函式 說明
scales::breaks_extended() 自動的「漂亮」刻度（ggplot2 預設）
scales::breaks_log() 適合對數轉換軸的刻度
scales::breaks_pretty() 為日期時間設計的美觀刻度
scales::breaks_width() 以固定寬度等距的刻度 
p_def <- base
p_ext <- base + scale_x_continuous(breaks = scales::breaks_extended())
p_few <- base + scale_x_continuous(breaks = scales::breaks_extended(n = 2))
p_def | p_ext | p_few

breaks_width() 以固定間隔放置刻度。一個選用的 offset 把所有刻度移動一個固定量：

p_w1 <- base + scale_x_continuous(breaks = scales::breaks_width(800))
p_w2 <- base + scale_x_continuous(breaks = scales::breaks_width(800, offset =  200))
p_w3 <- base + scale_x_continuous(breaks = scales::breaks_width(800, offset = -200))
p_w1 | p_w2 | p_w3

請注意，正的 offset 把所有刻度位置往上移，而負的 offset 把它們往下移。

次刻度

次刻度是出現在有標籤主刻度之間、淡淡的、無標籤的格線。它們透過 minor_breaks 引數設定，該引數同樣接受一個數值向量或函式。

次刻度在對數轉換軸上特別有資訊性，因為它們提供了間距非線性的清楚視覺提示：

# 跨越數個十進位級距建構一組密集的次刻度位置
mb <- unique(as.numeric(1:10 %o% 10^(0:3)))

base_log <- ggplot(toy, aes(log, const)) +
  geom_point() +
  labs(x = NULL, y = NULL) +
  scale_y_continuous(breaks = NULL)

p_no_mb <- base_log + scale_x_log10()
p_mb    <- base_log + scale_x_log10(minor_breaks = mb)
p_no_mb | p_mb

scales 套件也提供 scales::minor_breaks_n() 與 scales::minor_breaks_width() 作為以函式為基礎的替代。

標籤

每一個刻度都有一個相關聯的文字標籤。標籤由 labels 引數控制，它接受一個字元向量（每刻度一個標籤）或一個標示函式。

base_lbl <- ggplot(toy, aes(big, const)) +
  geom_point() +
  labs(x = NULL, y = NULL) +
  scale_y_continuous(breaks = NULL)

p_default <- base_lbl
p_custom  <- base_lbl +
  scale_x_continuous(breaks = c(2000, 4000), labels = c("2k", "4k"))
p_default | p_custom

scales 套件提供一系列現成的標籤格式器：

函式 輸出範例
scales::label_bytes() 1.5 kB、2.3 MB
scales::label_comma() 1,000
scales::label_dollar() $1,000
scales::label_ordinal() 1st、2nd、3rd
scales::label_percent() 10%
scales::label_pvalue() <.05、.34 
base_lbl2 <- ggplot(toy, aes(big, const)) +
  geom_point() +
  labs(x = NULL, y = NULL) +
  scale_x_continuous(breaks = NULL)

p_pct <- base_lbl2 + scale_y_continuous(labels = scales::label_percent())
p_eur <- base_lbl2 + scale_y_continuous(
  labels = scales::label_dollar(prefix = "", suffix = "€")
)
p_pct | p_eur

設定 breaks = NULL 與 labels = NULL 之間有一個重要的差別：

p_brk_null <- base_lbl2 + scale_y_continuous(breaks = NULL)
p_lbl_null <- base_lbl2 + scale_y_continuous(labels = NULL)
p_brk_null | p_lbl_null
  • breaks = NULL 同時移除刻度標記與主格線。
  • labels = NULL 只隱藏文字標籤，而保留刻度與格線。

轉換

預設情況下，連續位置尺度使用恆等（線性）轉換。覆寫這點是透過 trans 引數來做，它接受一個轉換名稱（字串）或一個來自 scales 套件的轉換器物件。

諸如 scale_x_log10()、scale_x_reverse() 與 scale_x_sqrt() 的便利包裝函式涵蓋了最常見的情況：

base_trans <- ggplot(mpg, aes(displ, hwy)) + geom_point()
p_rev_x <- base_trans + scale_x_reverse()
p_rev_y <- base_trans + scale_y_reverse()
base_trans | p_rev_x | p_rev_y

對於較不常見的轉換，直接使用 trans 引數：

p_recip <- ggplot(mpg, aes(displ, hwy)) +
  geom_point() +
  scale_y_continuous(trans = "reciprocal")

p_log <- ggplot(diamonds, aes(price, carat)) +
  geom_bin2d() +
  scale_x_continuous(trans = "log10") +
  scale_y_continuous(trans = "log10")

p_recip | p_log

scales 套件所提供的內建轉換完整清單：

名稱 函式 \(f(x)\) 反函式 \(f^{-1}(x)\)
"asn" \(\tanh^{-1}(x)\) \(\tanh(y)\)
"exp" \(e^x\) \(\log(y)\)
"identity" \(x\) \(y\)
"log" \(\log(x)\) \(e^y\)
"log10" \(\log_{10}(x)\) \(10^y\)
"log2" \(\log_2(x)\) \(2^y\)
"logit" \(\log\!\left(\frac{x}{1-x}\right)\) \(\frac{1}{1+e^y}\)
"probit" \(\Phi(x)\) \(\Phi^{-1}(y)\)
"reciprocal" \(x^{-1}\) \(y^{-1}\)
"reverse" \(-x\) \(-y\)
"sqrt" \(x^{1/2}\) \(y^2\)

你也可以直接呼叫轉換器物件——以下兩種寫法是等價的：

# 使用名稱字串
a <- ggplot(mpg, aes(displ, hwy)) +
  geom_point() +
  scale_y_continuous(trans = "reciprocal")

# 使用轉換器物件
b <- ggplot(mpg, aes(displ, hwy)) +
  geom_point() +
  scale_y_continuous(trans = scales::reciprocal_trans())

a | b

同樣地，便利包裝函式與明確的 trans 引數可互換：

# 透過便利包裝
a <- ggplot(diamonds, aes(price, carat)) +
  geom_bin2d() +
  scale_x_log10() +
  scale_y_log10()

# 透過 trans 引數
b <- ggplot(diamonds, aes(price, carat)) +
  geom_bin2d() +
  scale_x_continuous(trans = "log10") +
  scale_y_continuous(trans = "log10")

a | b

轉換尺度 vs. 轉換資料

一個重要的微妙之處：轉換尺度與直接轉換資料變數是不同的。

# 座標軸以*轉換後*（log10）的尺度標示
p_data <- ggplot(mpg, aes(log10(displ), hwy)) + geom_point() +
  ggtitle("aes() 中的 log10(displ)")

# 座標軸以*原始*資料尺度標示
p_scale <- ggplot(mpg, aes(displ, hwy)) + geom_point() +
  scale_x_log10() +
  ggtitle("scale_x_log10()")

p_data | p_scale

在兩種情況下，統計幾何都相同，但軸刻度標籤不同。轉換尺度（右）會在刻度上顯示原始資料值，這通常更易詮釋。重要的是，在兩種方法中，轉換都發生在任何統計摘要之前——如果需要在統計之後轉換，請使用 coord_trans()。

日期時間位置尺度

日期時間位置尺度

當一個日期或日期時間變數被映射到 x 或 y 時，ggplot2 自動選擇 scale_x_date() 或 scale_x_datetime() 作為預設尺度。這些尺度以對日期友善的引數擴展了連續尺度框架，以控制刻度與標籤。

支援的 R 日期／時間類別包括：

  • Date——日曆日期（基礎 R）
  • POSIXct——日期時間值（基礎 R）
  • hms——一天中的時刻值（hms 套件）

在繪圖前，用 as.Date()、as.POSIXct() 或 hms::as_hms() 把字元字串轉換成適當的類別。

日期尺度的刻度

date_breaks 引數用人類可讀的日期單位定位刻度標記："years"、"months"、"weeks"、"days"、"hours"、"minutes" 或 "seconds"。數字前綴設定間隔大小。

date_base <- ggplot(economics, aes(date, psavert)) +
  geom_line(na.rm = TRUE) +
  labs(x = NULL, y = NULL)

p_auto <- date_base
p_15yr <- date_base + scale_x_date(date_breaks = "15 years")
p_auto / p_15yr

date_breaks = "15 years" 是 breaks = scales::breaks_width("15 years") 的簡寫。使用較長的形式可存取 offset 引數：

# 刻度落在每月的第 9 日（薪資週期範例）
the_year  <- as.Date(c("2021-01-01", "2021-12-31"))
set_breaks <- scales::breaks_width("1 month", offset = 8)
set_breaks(the_year)
#>  [1] "2021-01-09" "2021-02-09" ... "2021-12-09" "2022-01-09"

日期尺度的次刻度

date_minor_breaks 引數控制次刻度間距的方式，與 date_breaks 控制主刻度間距的方式完全相同。

df <- data.frame(y = as.Date(c("2022-01-01", "2022-04-01")))
base_d <- ggplot(df, aes(y = y)) +
  labs(y = NULL) +
  theme_minimal() +
  theme(
    panel.grid.major = element_line(colour = "black"),
    panel.grid.minor = element_line(colour = "grey50")
  )

p_maj <- base_d + scale_y_date(date_breaks = "1 month")
p_min <- base_d + scale_y_date(
  date_breaks = "1 month",
  date_minor_breaks = "1 week"
)
p_maj | p_min

由於月份長度不一，次刻度（恰好相隔 7 天）會在每個月內產生略微不均的間距。

日期尺度的標籤

日期刻度標籤用 date_labels 引數格式化，它接受 strptime 風格的格式字串。

字串 意義
%S 秒（00–59）
%M 分（00–59）
%H 時，24 小時制（00–23）
%a / %A 縮寫／完整的星期名稱
%d 月中日（01–31）
%m 月，數字（01–12）
%b / %B 縮寫／完整的月份名稱
%y / %Y 不含／含世紀的年份 
base_econ <- ggplot(economics, aes(date, psavert)) +
  geom_line(na.rm = TRUE) +
  labs(x = NULL, y = NULL)

p_4digit <- base_econ + scale_x_date(date_breaks = "5 years")
p_2digit <- base_econ + scale_x_date(date_breaks = "5 years", date_labels = "%y")
p_4digit / p_2digit

換行字元 \n 可嵌入格式字串中，把標籤的各個成分跨多行堆疊，減少水平方向的擁擠：

lim <- as.Date(c("2004-01-01", "2005-01-01"))

p_compact  <- base_econ + scale_x_date(limits = lim, date_labels = "%b %y")
p_stacked  <- base_econ + scale_x_date(limits = lim, date_labels = "%B\n%Y")
p_compact | p_stacked

date_labels 的一個替代是把一個標示函式傳給 labels 引數。scales::label_date_short() 函式特別有用：它會自動縮減標籤，使重複的資訊（例如年份）在每個序列中只印一次，減少視覺雜亂：

lim <- as.Date(c("2004-01-01", "2005-01-01"))
base_econ +
  scale_x_date(
    limits = lim,
    labels = scales::label_date_short()
  )

離散位置尺度

離散位置尺度

當一個類別（因子或字元）變數被映射到 x 或 y 時，ggplot2 自動選擇 scale_x_discrete() 或 scale_y_discrete()。在內部，每個類別都被指派一個整數位置，這讓所有與寬度相關的參數（長條寬度、抖動高度等）都能以每類別單位區間的比例來指定。

# 透過標註揭露明確的整數映射
ggplot(mpg, aes(x = hwy, y = class)) +
  geom_point() +
  annotate("text", x = 5, y = 1:7, label = 1:7)

由於每個類別在離散尺度上占據單位寬度，在 geom_jitter() 中設定 height = .25 會產生橫跨類別箱 ±0.25 的抖動：

ggplot(mpg, aes(x = hwy, y = class)) +
  geom_jitter(width = 0, height = .25) +
  annotate("text", x = 5, y = 1:7, label = 1:7)

相同的邏輯支配 geom_boxplot() 中的盒寬：width = .4 為盒子保留類別槽位的 40%。

範圍界限、刻度與標籤

對於離散尺度，limits 定義的是完整、有序的可允許值集合（而非端點）。資料中存在但不在 limits 中的值會被無聲地捨棄；在 limits 中但資料中不存在的值會建立空白位置。

base_disc <- ggplot(toy, aes(const, txt)) +
  geom_label(aes(label = txt)) +
  scale_x_continuous(breaks = NULL) +
  labs(x = NULL, y = NULL)

p_extra  <- base_disc + scale_y_discrete(limits = c("a", "b", "c", "d", "e"))
p_reord  <- base_disc + scale_y_discrete(limits = c("d", "c", "a", "b"))
base_disc | p_extra | p_reord

breaks 引數選擇哪些值得到軸標籤；labels 用一個具名字元向量把特定值映射到自訂的顯示字串——而不擾動順序或刻度集合：

p_sel_brk <- base_disc + scale_y_discrete(breaks = c("b", "c"))
p_ren_lbl <- base_disc + scale_y_discrete(labels = c(c = "carrot", b = "banana"))
p_sel_brk | p_ren_lbl

scales::label_wrap() 函式對長類別名稱很有用——它會自動把文字在指定的字元寬度內折成多行。

標籤位置

軸標籤重疊是類別資料的一個常見問題。guides() 函式與 guide_axis() 對標籤放置給予精確的控制：

base_cat <- ggplot(mpg, aes(manufacturer, hwy)) + geom_boxplot()
base_cat

兩個有效的補救是錯位（把標籤錯開到多列）與旋轉：

p_dodge  <- base_cat + guides(x = guide_axis(n.dodge = 3))
p_rotate <- base_cat + guides(x = guide_axis(angle = 90))
p_dodge / p_rotate

guides() 是一個簡寫：等價的長形式把 guide 直接傳進尺度函式內。

# 等價的長形式
a <- base_cat + scale_x_discrete(guide = guide_axis(n.dodge = 3))
b <- base_cat + scale_x_discrete(guide = guide_axis(angle = 90))
a | b

分箱位置尺度

分箱位置尺度

分箱位置尺度占據連續與離散尺度之間的空間。它們把一個連續變數切成等距的箱，然後把每個箱當作一個離散類別。主要的函式是 scale_x_binned()（與 scale_y_binned()）。

最熟悉的應用是類直方圖的圖。下面的例子示範 geom_bar() 結合 scale_x_binned() 如何近似 geom_histogram()：

p_hist <- ggplot(mpg, aes(hwy)) + geom_histogram(bins = 8)
p_bin  <- ggplot(mpg, aes(hwy)) + geom_bar()  + scale_x_binned()
p_hist | p_bin

一個更實用的應用是把 scale_x_binned() 與 geom_count() 搭配。沒有分箱，對連續資料用 geom_count() 會產生過度擁擠的圖，因為每一對相異的座標都會得到自己的點：

ggplot(mpg, aes(displ, hwy)) + geom_count()

在兩軸都加上分箱尺度，會把鄰近的點分組到離散的儲存格中，產生一個可讀得多的摘要：

ggplot(mpg, aes(displ, hwy)) +
  geom_count() +
  scale_x_binned(n.breaks = 15) +
  scale_y_binned(n.breaks = 15)

n.breaks 引數控制切點的數目。非位置美學（顏色、大小等）的分箱尺度在 §11.4 與 §12.1.2 討論。
